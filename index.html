Cursos

Concluir o curso "Como Criar um Site Simples usando HTML, CSS e JavaScript". Foi uma experiência enriquecedora que me proporcionou uma base sólida no desenvolvimento web. Durante o curso, tive a oportunidade de aprender e aplicar conceitos essenciais das principais tecnologias de front-end. No módulo de HTML, adquiri conhecimentos para estruturar páginas da web de maneira organizada e semântica, criando elementos como títulos, parágrafos, links, tabelas e formulários. Já em CSS, explorei técnicas para estilizar páginas, utilizando cores, fontes, espaçamentos e layouts responsivos, garantindo um design visualmente atraente e funcional. Por fim, em JavaScript, aprendi a tornar os sites interativos, adicionando funcionalidades dinâmicas, como validação de formulários e manipulação de elementos do DOM.
